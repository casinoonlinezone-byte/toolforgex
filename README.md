## Quick start

1. Serve files with a local static server (Live Server, python -m http.server, or 
px serve).
2. Open http://127.0.0.1:8080 (or the server URL) in your browser.

## Deploy

1. Push to GitHub
2. Connect repository in Vercel
3. Add Firebase config (see js/firebase.sample.js -> copy to js/firebase.js)
